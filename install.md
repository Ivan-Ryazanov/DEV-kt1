# Установка и запуск проекта Drive Simulation

Этот документ содержит инструкции по установке и запуску проекта **Drive Simulation** на вашей машине.

## Требования

- [.NET Core SDK](https://dotnet.microsoft.com/download) версии 6.0 или выше
- Текстовый редактор или IDE (например, Visual Studio, Visual Studio Code)

## Шаги по установке

### 1. Скачать репозиторий

Клонируйте проект с GitHub или скачайте его в виде ZIP-архива и распакуйте:

```bash
git clone https://github.com/ваш_пользователь/ваш_репозиторий.git
```

### 2. Открытие проекта

- Откройте папку проекта в вашей IDE или текстовом редакторе.
- Если используете Visual Studio, откройте файл `.sln`.
- В Visual Studio Code откройте папку проекта.

### 3. Восстановление зависимостей и сборка

В командной строке выполните команды:

```bash
dotnet restore
dotnet build
```

или используйте встроенные функции IDE для сборки проекта.

### 4. Запуск проекта

Для запуска проекта выполните команду:

```bash
dotnet run
```

или запустите программу через IDE.

## Использование

После запуска программа выполнит демонстрационный сценарий, показывающий работу класса `Driver` и его методов.
