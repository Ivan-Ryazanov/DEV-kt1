# Drive Simulation Project

Этот проект представляет собой симуляцию движения автомобиля с учетом его технического состояния и маршрута. В нем реализована классическая модель водителя, который может двигаться вперед и назад, а также отслеживать свое местоположение и состояние автомобиля.

## Основные возможности

- Создание объекта водителя с заданной средней скоростью и начальной позицией.
- Передвижение вперед и назад по трассе с учетом времени.
- Автоматическое снижение технического состояния автомобиля в зависимости от пройденного расстояния.
- Определение текущего местоположения (город A, B, C или трасса).
- Вывод текущего технического состояния автомобиля.
- Обработка ситуации, когда автомобиль находится в неисправном состоянии (Condition <= 0).

## Структура проекта

### Класс `Driver`

Этот класс моделирует водителя и его автомобиль. Основные свойства и методы:

- `AverageSpeed` — средняя скорость автомобиля.
- `Coordinates` — текущие координаты автомобиля.
- `Condition` — техническое состояние автомобиля (в процентах).
- `Drive(double time)` — движение вперед за указанное время (часы).
- `DriveBack(double time)` — движение назад за указанное время.
- `PrintLocation()` — выводит текущую локацию автомобиля.
- `PrintCarTechnicalState()` — отображает текущий уровень технического состояния.

### Основная программа (`Program`)

В программе демонстрируются различные сценарии использования класса `Driver`:

- Начальное состояние автомобиля.
- Поездка вперед и назад с отображением результатов.
- Длинная поездка.
- Попытка движения неисправным автомобилем.

## Использование

Для запуска проекта выполните сборку и запуск программы. В консоли вы увидите пошаговый вывод состояния автомобиля и его положения после каждой операции.
